0.5 Vector calculus

0.5.1 Longitudinal and transversal projection of vectors

A vector 𝒂 can be split into two parts: the longitudinal part 𝒂∥, which is parallel to another vector 𝒃 and the transversal part 𝒂⟂, which is perpendicular to 𝒃. The length of the longitudinal part a∥ and the transversal part a⟂ can be computed from geometry (see figure LABEL:fig:projection)

a∥a=cos⁡α ⇒a∥=a⁢cos⁡α=a⁢b⁢cos⁡αb=𝒂⋅𝒃b, (36)
a⟂a=sin⁡α ⇒a⟂=a⁢sin⁡α=a⁢b⁢sin⁡αb=|𝒂×𝒃|b. (37)

But from the Pythagorean theorem we can get another expression for the length of the transversal part

a2=a∥2+a⟂2 ⇒a⟂2=a2-a∥2=a2-(𝒂⋅𝒃)2b2. (38)

Substituting equation (37) in equation (38) we get

(|𝒂×𝒃|)2b2=a2-(𝒂⋅𝒃)2b2,

which leads us to the following expression for the square of the norm of the cross product

|𝒂×𝒃|2=(a⁢b)2-(𝒂⋅𝒃)2. (39)

This is again Lagrange’s identity (see (7)).

0.5.2 Vector identities

In this chapter we show the derivation of some vector quantities in cartesian tensor notation.

(𝒖⋅∇)⁢𝒗

For some arbitraty vectors ui,vi we can write

uj⁢∂∂⁡rj⁢vi =uj⁢∂∂⁡ri⁢vj-uj⁢∂∂⁡ri⁢vj⏞0+uj⁢∂∂⁡rj⁢vi
=uj⁢∂∂⁡ri⁢vj-δi⁢k⁢δj⁢l⁢uj⁢∂∂⁡rk⁢vl+δi⁢l⁢δj⁢k⁢uj⁢∂∂⁡rk⁢vl
=uj⁢∂∂⁡ri⁢vj-(δi⁢k⁢δj⁢l-δi⁢l⁢δj⁢k)⁢uj⁢∂∂⁡rk⁢vl
=uj⁢∂∂⁡ri⁢vj-ϵm⁢i⁢j⁢ϵm⁢k⁢l⁢uj⁢∂∂⁡rk⁢vl
=uj⁢∂∂⁡ri⁢vj-ϵi⁢j⁢m⁢uj⁢ϵm⁢k⁢l⁢∂∂⁡rk⁢vl.

In vector notation this can be expressed like

(𝒖⋅∇)⁢𝒗=𝒖⋅(∇⁡𝒗)-𝒖×(∇×𝒗) (40)

(𝒗⋅∇)⁢𝒗

Inserting uj=vj into equation (40) yields

vj⁢∂∂⁡rj⁢vi=vj⁢∂∂⁡ri⁢vj-ϵi⁢j⁢m⁢vj⁢ϵm⁢k⁢l⁢∂∂⁡rk⁢vl

For vj⁢∂∂⁡ri⁢vj we can write

vj⁢∂∂⁡ri⁢vj =∂∂⁡ri⁢(vj⁢vj)-vj⁢∂∂⁡ri⁢vj

and therefore

vj⁢∂∂⁡ri⁢vj=∂∂⁡ri⁢(12⁢vj⁢vj).

Using this we get for

vj⁢∂∂⁡rj⁢vi=∂∂⁡ri⁢(12⁢vj⁢vj)-ϵi⁢j⁢m⁢vj⁢ϵm⁢k⁢l⁢∂∂⁡rk⁢vl

or in vector notation

(𝒗⋅∇)⁢𝒗=12⁢∇⁡𝒗2-𝒗×(∇×𝒗) (41)

∇×(𝒖×𝒗)

The i-th component of the rotation of a cross product of two vectors is

[∇×(𝒖×𝒗)]i =ϵi⁢j⁢k⁢∂∂⁡rj⁢ϵk⁢l⁢m⁢ul⁢vm
=ϵk⁢i⁢j⁢ϵk⁢l⁢m⁢∂∂⁡rj⁢(ul⁢vm)
=(δi⁢l⁢δj⁢m-δi⁢m⁢δj⁢l)⁢∂∂⁡rj⁢(ul⁢vm)
=∂∂⁡rj⁢(ui⁢vj)-∂∂⁡rj⁢(uj⁢vi)
=ui⁢∂∂⁡rj⁢vj+vj⁢∂∂⁡rj⁢ui-uj⁢∂∂⁡rj⁢vi-vi⁢∂∂⁡rj⁢uj

It can be written in vector notation like

∇×(𝒖×𝒗)=𝒖⁢(∇⋅𝒗)-𝒗⁢(∇⋅𝒖)+(𝒗⋅∇)⁢𝒖-(𝒖⋅∇)⁢𝒗 (42)

0.5.3 Jacobian determinant

Definition

If the cartesian coordinates are given as functions of general curvilinear coordinates a1,a2,a3, i.e.

x=x⁢(a1,a2,a3), y=y⁢(a1,a2,a3), z=z⁢(a1,a2,a3), (43)

it follows for the infinitesimal volume element in curvilinear coordinates

d⁢V=|J|⁢d⁢a1⁢d⁢a2⁢d⁢a3. (44)

Thereby the Jacobian determinant J is defined as

J=|∂⁡(x,y,z)∂⁡(a1,a2,a3)|=|∂⁡x∂⁡a1∂⁡x∂⁡a2∂⁡x∂⁡a3∂⁡y∂⁡a1∂⁡y∂⁡a2∂⁡y∂⁡a3∂⁡z∂⁡a1∂⁡z∂⁡a2∂⁡z∂⁡a3|=∑i,j,kϵi⁢j⁢k⁢∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak. (45)

Time derivative of Jacobian determinant

With vi=d⁢xid⁢t it follows for the time derivative of J

d⁢Jd⁢t =dd⁢t⁢∑i,j,kϵi⁢j⁢k⁢∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak (46)
=∑i,j,kϵi⁢j⁢k⁢(∂⁡vx∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak+∂⁡x∂⁡ai⁢∂⁡vy∂⁡aj⁢∂⁡z∂⁡ak+∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡vz∂⁡ak). (47)

From ∂⁡vk∂⁡ai=∑l∂⁡vk∂⁡xl⁢∂⁡xl∂⁡ai (k∈x,y,z) we get:

d⁢Jd⁢t=∑i,j,k,lϵi⁢j⁢k⁢(∂⁡vx∂⁡xl⁢∂⁡xl∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak+∂⁡x∂⁡ai⁢∂⁡vy∂⁡xl⁢∂⁡xl∂⁡aj⁢∂⁡z∂⁡ak+∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡vz∂⁡xl⁢∂⁡xl∂⁡ak). (48)

Analysis of the first term in the bracket for

l =1:∂⁡vx∂⁡x⁢∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak,
l =2:∂⁡vx∂⁡y⁢∂⁡y∂⁡ai⁢∂⁡y∂⁡aj⏟symmetric⁢∂⁡z∂⁡ak⁢ is (multiplied with ϵi⁢j⁢k) 0 when summed up,
l =3:∂⁡vx∂⁡z⁢∂⁡z∂⁡ai⏞⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak⏞⁢ is (multiplied with ϵi⁢j⁢k) 0 when summed up.

So for symmetry reasons most of the contributions are zero when summed up. In we end we see: For the first term there is only a non-vanishing contribution for l=1, for the second term there is only one for l=2 and for the third term there is only one for l=3. So we get

d⁢Jd⁢t=∑i,j,kϵi⁢j⁢k⁢(∂⁡vx∂⁡x+∂⁡vy∂⁡y+∂⁡vz∂⁡z)⁢∂⁡x∂⁡ai⁢∂⁡y∂⁡aj⁢∂⁡z∂⁡ak=(∇⋅𝒗)⁢J. (49)
