Number inequality

We start with the obvious inequality that the absolute value of a number |a| is always greater or equal the number a itself

|a|≥a (1)

Using this we can prove that for numbers a and b we have

|a|+|b|≥|a+b| (2)

because

(|a|+|b|)2 ≥(|a+b|)2 (3)
|a|2+2⁢|a|⁢|b|+|b|2 ≥|a2+2⁢a⁢b+b2| (4)
2⁢|a|⁢|b| ≥2⁢|a⁢b| (5)

The last line is always true if |a|≥a, which completes our prove.

0.1.2 Cauchy-Schwarz inequality

Because sin2⁡(ϕ)+cos2⁡(ϕ)=1 we have

|a|2⁢|b|2=|a|2⁢|b|2⁢(sin2⁡(ϕ)+cos2⁡(ϕ)) (6)

With the definition of the scalar product |a⋅b|=|a|⁢|b|⁢cos⁡(ϕ) and the cross product |a×b|=|a|⁢|b|⁢sin⁡(ϕ) of vectors a and b we can write this expression as

|a|2⁢|b|2=|a×b|2+|a⋅b|2 (7)

This is called Lagrange’s identity. Since all terms are squared and therefor positive we immediatelly can derive the inequalities

|a|2⁢|b|2 ≥|a⋅b|2 (8)
|a|2⁢|b|2 ≥|a×b|2 (9)

The first of these equations is the Cauchy-Schwarz inequality. The second equations doesn’t seem to have a name in the literature.

The Cauchy-Schwarz inequality comes in many different forms. For example for n-dimensional vectors it can be written in cartesian coordinates like

∑(ai)2⁢∑(bi)2≥(∑ai⁢bi)2 (10)

This can even be generalized to uncountable infinite dimensional vectors (also called continuous square integrable functions) like

(∫|a⁢(x)|2⁢𝑑x)⋅(∫|b⁢(x)|2⁢𝑑x)≥|∫a⁢(x)⋅b⁢(x)⁢𝑑x|2 (11)
